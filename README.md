# 한지선 포트폴리오

프론트엔드 개발자 한지선의 포트폴리오 웹사이트입니다.

## 🚀 기술 스택

- **Frontend**: React 19, TypeScript
- **Styling**: Emotion (CSS-in-JS)
- **State Management**: Zustand
- **Animation**: Framer Motion
- **Icons**: React Icons
- **Build Tool**: Vite
- **Routing**: React Router DOM

## 📋 프로젝트 구조

```
src/
├── components/     # 재사용 가능한 컴포넌트
├── pages/         # 페이지 컴포넌트
├── styles/        # 글로벌 스타일 및 테마
├── types/         # TypeScript 타입 정의
├── hooks/         # 커스텀 훅
└── utils/         # 유틸리티 함수
```

## 🎯 주요 기능

- **반응형 디자인**: 모바일, 태블릿, 데스크톱 대응
- **다크/라이트 모드**: 테마 전환 기능
- **애니메이션**: Framer Motion을 활용한 부드러운 애니메이션
- **SEO 최적화**: 메타 태그 및 구조화된 데이터
- **접근성**: ARIA 라벨 및 키보드 네비게이션 지원

## 🛠️ 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```
